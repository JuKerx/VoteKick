# VoteKick
A simple plugin with a lot of customization and permissions per command, Supports 1.19 up to 1.20.x (Bukkit, Spigot, Paper, Purpur and Folia)

## Commands

- **/votekick** or **/vk**
  - List of commands for the plugin.
- **/votekick start (player)**
- **/votekick yes**
- **/votekick no**
- **/votekick check**
- **/votekick cancel**
- **/votekick reload**
